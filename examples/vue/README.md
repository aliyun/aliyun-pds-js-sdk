# PDS JS SDK Demo

power by `Vue 3 + koa2`

## 目录结构

```
example/vue/
  |-- /lib    // 后端 koa 代码
    |-- /config
      |-- conf.js // 配置
  |-- /src    // 前端 vue 代码
```

注意: 启动前准备好 lib/config/conf.js 内容。详情参考 lib/config/index.js。


## 启动命令

```bash
npm start
```

## 说明

本示例代码仅包含最基本的文件列表展示和上传下载逻辑，其他功能暂未集成。
